# 正式リリース前チェックリスト（Issue #150）

- 最終更新: 2026-05-18
- 対象: 正式リリース候補（非公式ファンツール）
- 注意: 本チェックリストは公開判断のための運用資料であり、法務判断の確定は行わない（要確認）。

## 0. リリース判定の前提

- [ ] 非公式ファンツールであることを主要導線（ヘッダー/フッター/主要ページ）で再確認
- [ ] 正確性非保証・OCRは入力補助である旨を再確認
- [ ] OCR画像はサーバー保存しない方針を再確認
- [ ] 低信頼度OCR候補を自動確定しない方針を再確認
- [ ] 保存/共有/ランキングpayloadにOCRメタ情報（raw text/confidence/candidates等）を混入しない方針を再確認

## 1. Release（公開設定・環境変数・デプロイ前）

- [ ] 本番URLを確定し、公開記録へ反映（README/運用メモ等）
- [ ] `NEXT_PUBLIC_SITE_URL` が本番URLと一致している
- [ ] `NEXT_PUBLIC_ROBOTS_NOINDEX=false`（本番公開時）を確認
- [ ] `robots.txt` と `sitemap.xml` が本番方針（index/noindex）と矛盾しない
- [ ] title/description/canonical/OGPの主要ページ整合を確認
- [ ] noindex運用解除時の反映確認手順（再build/再deploy/再確認）を記録

## 2. Security

- [ ] `.env.local` がコミットされていない
- [ ] Supabase公開キーは Anon Key のみ（service role key不使用）
- [ ] 保存系payloadにアクセストークン・secret・メール等の不要情報が含まれない
- [ ] 依存関係監査（`pnpm audit` 等）の実施可否と結果を記録
- [ ] 既知脆弱性の対応方針（許容/修正予定）を記録

## 3. Privacy

- [ ] OCR画像はブラウザ内処理を基本とし、サーバー永続保存しない説明が表示される
- [ ] 個人情報・アカウント情報・機密情報画像のアップロード禁止を明記
- [ ] プライバシー方針ページへの導線（フッター等）が有効
- [ ] OCR補助であることと最終確認責任（利用者）を明記

## 4. License / Legal（要確認）

- [ ] 利用規約・免責・プライバシー方針の3ページが公開状態で導線有効
- [ ] 公式名称・略称・ゲーム内用語の利用可否は「要確認」扱いを維持
- [ ] 公式素材・ゲーム画像・ロゴ等の利用可否は「要確認」扱いを維持
- [ ] 商用利用前の専門確認未完了項目をリリースノート/運用記録へ明記

## 5. Support / 運用

- [ ] 問い合わせ導線（`/contact`）がアクセス可能
- [ ] サポート範囲（回答保証なし・限定対応）を規約系文書と整合
- [ ] 障害時アナウンス方法（Issue/Discussion等）を記録
- [ ] 更新履歴掲載先（`/updates` または release-notes）を確定

## 6. 機能・品質

- [ ] `/score` `/card` `/compare` の主要導線が動作
- [ ] 共有URL復元が動作
- [ ] ビルドカード生成・保存・共有が動作
- [ ] OCR未使用時の手動入力導線が継続動作
- [ ] 低信頼度候補で手動補正導線とエラー表示が機能
- [ ] fallback設定時でも `/score` `/card` `/compare` がクラッシュしない

## 7. テスト実行記録

- [ ] `pnpm lint` 実行結果を記録
- [ ] `pnpm test` 実行結果を記録
- [ ] `pnpm build` 実行結果を記録
- [ ] 未実行の場合は理由（docs中心作業/環境要因）を記録

## 8. Rollback / Post-release

### 8.1 Rollback手順（事前確認）

- [ ] rollback責任者を確定
- [ ] 直前安定版コミット/タグを記録
- [ ] 問題発生時は `git revert <release_commit>` または直前タグ再deploy手順を確認
- [ ] noindex再有効化が必要な場合、`NEXT_PUBLIC_ROBOTS_NOINDEX=true` への切替手順を確認

### 8.2 Post-release確認項目

- [ ] 本番トップと主要導線の表示崩れなし
- [ ] 利用規約/免責/プライバシー方針リンクが機能
- [ ] robots/sitemap/canonicalが想定どおり
- [ ] お問い合わせ導線が機能
- [ ] 重大問い合わせ/障害の初動対応を記録

## 9. リリース判定

- [ ] Go / No-Go 判定会議（人間最終判断）を実施
- [ ] 未解決の「要確認」事項を列挙し、公開可否判断に反映
- [ ] 判定結果・日時・担当者をRepositoryに記録
