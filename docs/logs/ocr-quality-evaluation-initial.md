## 6. 初回評価からの改善候補（優先順位）

### P1（次Issueで着手）
1. 低解像度・ぼやけ画像での名称認識低下対策（前処理ガイド/撮影ガイド文言の最適化）。
2. 韓国語表示時のマッピング誤り縮小（alias辞書運用・同義語レビュー手順強化）。
3. 低スペック端末向けの待機時間期待値表示改善（4秒超過時の明示強化）。

### P2
4. 暗い背景での補正率増加に対するUI補助文言の改善。
5. 低信頼度候補が連続するケースの補正導線簡略化（手動確認チェックポイント再配置）。

### P3
6. カテゴリ別サンプル母数拡張（週次で各カテゴリ20件以上を目標化）。
