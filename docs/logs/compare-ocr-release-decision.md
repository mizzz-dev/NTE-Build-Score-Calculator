# `/compare` OCR正式展開可否 最終判定ログ（Issue #125）

- 作成日: 2026-05-15
- 対象Issue: #125
- 関連Issue: #113, #115, #117, #119, #121, #123
- 関連PR: #124（第3サイクルKPI計測完了）

## 1. 結論

- 最終判定: **条件付き正式展開可**
- 判定区分: 「正式展開可 / 条件付き正式展開可 / 追加改善後に再判定 / rollback候補」のうち **条件付き正式展開可**

## 2. 第1〜第3サイクル KPI推移

| KPI | 第1サイクル | 第2サイクル | 第3サイクル | 傾向 |
|---|---:|---:|---:|---|
| OCR処理時間 p95（全体） | 3.9秒 | 3.7秒 | 3.6秒 | 改善 |
| OCR処理時間 p95（低性能端末） | 4.2秒 | 4.0秒 | 3.8秒 | 改善 |
| OCR処理時間 p95（中性能端末） | 3.7秒 | 3.5秒 | 3.4秒 | 改善 |
| OCR処理時間 p95（高性能端末） | 3.2秒 | 3.0秒 | 2.9秒 | 改善 |
| A/B取り違え誤反映率 | 0.0% | 0.0% | 0.0% | 維持 |
| 未確定項目残存率 | 14.1% | 12.6% | 11.4% | 改善 |

| fallback率 | 9.4% | 8.6% | 7.9% | 改善 |
| 手動補正率 | 19.2% | 17.8% | 16.4% | 改善 |

## 3. 判定理由

1. 重点KPI（低性能端末p95 / fallback率 / 比較前確認離脱率）が3サイクル連続で改善しており、悪化トレンドがない。
2. 安全性KPI（A/B取り違え誤反映率）は3サイクル連続で0.0%を維持。
3. 保存payload・共有URL・ランキングpayload互換を維持し、OCRメタ非混入を継続。
4. 未確認OCR値の比較計算混入は3サイクルで0件。
5. 一方で `image_quality_low` と `unresolved_items_remaining` が主因として残るため、監視しきい値とrollback条件を付与して正式展開する。

## 4. 前提・制約

- OCRアルゴリズムは変更しない。
- DB / auth / infra / deployment は変更しない。
- 保存payload仕様 / ランキング仕様は変更しない。
- 画像保存は追加しない。
- 外部OCR API連携は追加しない。
- `/card` 既存OCR導線は不要改修しない。

## 5. 残リスク

1. 低品質画像（`image_quality_low`）由来でfallbackが増加する可能性。
2. `unresolved_items_remaining` 由来の確認離脱が再上昇する可能性。
3. 低性能端末の一時的な性能変動によるp95悪化リスク。

## 6. 正式展開後の監視・対応方針（要約）

- 監視KPI: p95（全体/端末別）、A/B誤反映率、未確定残存率、比較前離脱率、fallback率、手動補正率。
- 超過時対応: 警戒しきい値超過で運用ログ記録 + 改善Issue起票、rollbackしきい値超過で即導線無効化。
- 詳細手順: `docs/runbooks/compare-ocr-release-runbook.md` を参照。

## 7. `/compare` OCR導線無効化時の確認

- OCR導線非表示、手動入力導線有効、比較前確認ガード維持、OCRメッセージ非表示、比較結果の手動入力整合を確認する。

## 8. 最低回帰確認（互換性）

- 保存payloadにOCRメタ（`confidence` 等）が混入しないこと。
- 共有URLのキー構造・エンコード仕様が不変であること。
- ランキングpayloadの構造・型・必須項目が不変であること。

## 9. ADR要否判断

- 判定: **ADR不要**
- 理由:
  1. アーキテクチャ・データ設計・デプロイ構成の変更がない。
  2. 既存方針の運用判定を確定したドキュメント変更に留まる。
  3. 将来監査に必要な判断根拠は本ログとRunbookへ記録済み。

## 10. security / privacy / license / release / support 観点

- security: OCRメタは永続payloadへ保存しない。
- privacy: 評価画像・PIIをRepositoryへ保存しない。
- license: 非公式ファンツールであり、公式素材利用可否は要確認。
- release: 条件付き正式展開可（監視・rollback条件付き）。
- support: OCR導線無効化時も手動入力で比較機能継続可能。
