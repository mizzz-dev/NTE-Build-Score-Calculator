# スコア計算仕様（初期方針）

## 1. 目的
ビルド品質を定量評価し、比較可能な形式で提示する。

## 2. 計算対象
- 装備カテゴリ: Cartridge / Module / Gear / Console
- 評価単位:
  - 装備1個単位
  - キャラ全体ビルド（複数装備合算）

## 3. 評価軸
- ロール別評価
- キャラ別評価
- メインステータス一致
- サブステータス重み
- セット効果・装備効果

## 4. スコアモデル方針
- 「攻撃力 + 会心率 + 会心ダメージ」型の複合式を基本思想とする
- 点数スケールは固定上限より、ゲーム仕様追従性と比較可能性を優先
- ランクは10段階程度（最大 SS）で閾値管理する

## 5. 拡張性要件
- 重み・閾値・評価対象は設定データ化
- ゲームアップデート時に管理画面/設定更新で追従可能
- ユーザーのカスタム重み調整を将来拡張可能な構造

## 6. 入力制約（方針）
- ステータスは `%` と固定値を区別
- 同一サブステータス重複は原則禁止
- 最小値/最大値/小数点桁数はゲーム仕様準拠

## 7. 要確認事項
- 正式なダメージ計算式
- キャラ別最適ステータス定義
- セット効果の加点ロジック
- ステータス上限・丸め規則
