# definition-of-done

## 通常完了条件
- 実装完了
- pnpm lint 成功
- pnpm test 成功
- pnpm build 成功
- docs更新
- 作業ログ保存
- AIプロンプトログ保存
- PR本文日本語
- commit message日本語
- 必要なADR保存
- rollback確認
- branch cleanup確認
- handoff保存

## 商用完了条件
通常完了条件に加えて:
- 非公式ファンツール影響確認
- 公式素材利用影響確認
- license確認
- security確認
- privacy確認
- Supabase / 認証影響確認
- 環境変数漏洩確認
- release readiness確認
- support note要否確認
- incident response確認
- monitoring要否確認
- customer impact確認
