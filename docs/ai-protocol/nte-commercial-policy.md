# NTE商用運用ポリシー

## 1. 非公式ファンツール表記
- 非公式ファンツールであることを明記する。
- 公式との関係を誤認させる表現を禁止する。
- 公式素材利用可否は「要確認」とし、商用利用前に権利確認する。

## 2. 権利・ライセンス
確認対象: 公式画像、ゲーム内スクリーンショット、ロゴ、キャラクター名、アイテム名、数値データ、OCR対象画像、ユーザー投稿画像。

## 3. Supabase / 認証
- 公開可能なのはAnon Keyのみ。
- service role keyをフロントエンド・Repository・ログに出さない。
- ユーザーデータ最小化、削除依頼対応方針を整備。
- ログイン情報をログへ出力しない。

## 4. OCR / 画像解析
- tesseract.js利用時、画像とOCR結果の保存有無を事前定義する。
- 個人情報・機密情報混入リスクを考慮する。
- クライアント処理/サーバー処理の責務を明確化する。

## 5. 商用公開前チェック
- 利用規約
- プライバシーポリシー
- 権利確認
- サポート窓口
- 障害対応
- リリースノート
- rollback
- monitoring
