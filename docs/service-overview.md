# サービス概要

## 1. サービス名
- **NTE Build Score Calculator**

## 2. サービス目的
Neverness to Everness（以下、NTE）向けの**非公式ファンツール**として、ユーザーが装備・キャラ・ロール・ステータス情報を入力し、ビルドの評価・比較・共有・保存を行えるようにする。

## 3. 対象ユーザー
- 主対象: 日本語ユーザー
- 副対象: 英語ユーザー、韓国語ユーザー

## 4. 提供価値
- 装備単位とビルド全体のスコア可視化
- キャラ別・ロール別の評価支援
- ビルド比較、ビルドカード生成、SNS共有
- URL共有による入力復元
- ログインユーザーの保存・ランキング参加
- ゲストユーザーの簡易利用と履歴保持

## 5. 対応プラットフォーム
- スマートフォン
- タブレット
- PC

## 6. UI / UX 方針
- ダーク・ネオン・近未来都市UI
- モバイルファーストで設計
- トップからスコア計算へ最短導線

## 7. サービス区分と法務方針
- 公式サービスではなく、NTEの**非公式ファンツール**として提供する。
- 利用規約・著作権・ガイドラインを尊重し、違反リスクを避ける運用とする。
- 公式画像・ロゴ・キャラ画像の利用は「要確認」とし、許諾・規約確認後に反映可否を決定する。

## 8. 初期リリースで提供する大分類
- スコア計算（装備単位 / ビルド全体）
- ビルド比較
- ビルドカード生成（画像保存・共有）
- 保存機能（ログイン / ゲスト）
- ランキング
- OCR / スクリーンショット読取
- ダメージシミュレーター
- 管理画面
- 多言語対応（日本語・英語・韓国語）
