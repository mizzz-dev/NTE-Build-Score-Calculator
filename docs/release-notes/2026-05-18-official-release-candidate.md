# 正式リリース候補リリースノート（Issue #150）

- 作成日: 2026-05-18
- 対象: 正式リリース候補
- 位置づけ: 非公式ファンツール
- 注意: 法務判断は本書で確定しない（要確認）。

## 1. リリース概要

OCR入力補助を `/score` `/card` `/compare` へ展開した現行機能セットを、正式公開候補として整理した。
公開前SEO最小整備（metadata/canonical/robots/sitemap）と規約導線（利用規約・免責・プライバシー方針）を前提に、公開判断に必要な確認事項をドキュメント化した。

## 2. 主要機能

- スコア計算（設定駆動）
- OCR入力補助（`/score` `/card` `/compare`）
- 共有URL復元
- ビルドカード生成・保存・共有
- 比較表示（A/B）
- ランキング関連導線

## 3. 公開範囲

- 主要公開ページ: `/` `/score` `/card` `/compare` `/ranking` `/presets` `/statuses` `/guide` `/faq` `/updates` `/contact` `/terms` `/privacy` `/disclaimer`
- 非公開/対象外: 管理系ページ、本番公開判断未了の導線

## 4. 既知制限

1. OCRは入力補助であり、最終確定は利用者の手動確認が必須。
2. OCR結果・計算結果・ランキング結果の正確性は保証しない。
3. 公式名称・略称・ゲーム内用語・公式素材の利用可否は要確認。
4. 商用利用前の専門確認（権利・法務）は未完了。

## 5. 非公式ファンツール表記

- 本サービスは非公式ファンツールであり、公式運営・権利者とは関係しない。
- 誤認防止のため、主要導線で非公式表記を維持する。

## 6. セキュリティ/プライバシー方針（公開前確認）

- OCR画像はブラウザ内処理を基本とし、サーバー永続保存しない方針。
- 個人情報・アカウント情報・機密情報を含む画像アップロードは禁止。
- 保存/共有/ランキングpayloadへOCRメタ情報（raw text/confidence/candidates等）を混入しない方針。
- Supabase公開設定はAnon Keyのみ。service role keyは使用しない。

## 7. リリース設定確認

- 本番URLを確定して `NEXT_PUBLIC_SITE_URL` と一致させる。
- 本番公開時は `NEXT_PUBLIC_ROBOTS_NOINDEX=false` を確認する。
- robots/canonical/sitemap の整合を公開直前に再確認する。

## 8. サポート範囲

- 問い合わせ導線は ` /contact ` を利用。
- 回答時期・解決を保証しない限定サポート。
- 重大障害時は Issue/Discussion 等で告知する運用を基本とする。

## 9. Breaking Change

- Breaking change: **なし**（本リリース候補は既存機能の公開準備整理が中心）。

## 10. Rollback方針

1. 問題発生時は直前安定コミットへ `git revert` または直前タグ再deployで切り戻す。
2. 必要時は `NEXT_PUBLIC_ROBOTS_NOINDEX=true` に戻し、検索露出を一時停止する。
3. 影響範囲（/score /card /compare /規約導線）を優先確認し、復旧状況を記録する。

## 11. 残課題（正式公開前 / 商用利用前）

- 商用利用前の専門確認（名称・素材・免責文言・収益化導線）の完了。
- 公開時の本番URL最終確定と環境変数最終監査。
- リリース判定会議での Go / No-Go 最終記録。
