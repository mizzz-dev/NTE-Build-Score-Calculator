## 0. 記入ルール（必読）

1. 未確認項目を推測で埋めない。
   - 不明な項目は `未確認` と記載し、補完予定日・担当者を明記する。
2. 本テンプレートへの記録者は「AI補助可 / 最終承認と最終実行は人間のみ」を維持する。
3. 法務判断（公式素材利用可否・商用利用可否など）は本テンプレートで確定しない。
   - ステータスは `要確認` を維持する。
4. 機微情報（トークン、秘密鍵、個人情報、ローカルファイルパス、内部管理画面の秘匿情報）は記録しない。
5. URL・時刻は可能な限り一次情報にリンクし、時刻はUTCで併記する。

## 6. 証跡として保存してよい情報 / 保存してはいけない情報

### 6.1 保存してよい情報（Repository保存可）

- Issue / PR / Discussion / Runbook / 運用ログへの公開URL
- 判定結果（Go/No-Go/保留）
- 承認者の役割名または公開可能な識別名（個人情報を含まない形）
- UTC時刻
- 対象コミットSHA
- `NEXT_PUBLIC_SITE_URL` 一致可否、`NEXT_PUBLIC_ROBOTS_NOINDEX` 値の判定結果
- deployログの公開可能URL（秘密情報を含まないもの）
- rollback要否判定と理由（機微情報を除く）

### 6.2 保存してはいけない情報（Repository保存禁止）

- APIキー、トークン、秘密鍵、パスワード、Cookie、セッション情報
- `service role key` を含む秘匿環境変数
- 個人情報（メール、電話番号、住所、個人名フルネーム等）
- ローカル環境固有ファイル（`.env.local`、端末固有パス、個人スクリーンショット原本）
- 内部管理画面URLのうち公開不可情報を含むもの
- 法務判断を確定した断定表現（未確認事項の断定）
